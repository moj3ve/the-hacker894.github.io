Description

  • Aye matey! Ever wanted to have Sileo in ye native tongue? Well this tweak be for ya, lad.

  • Run sileopirate in MobileTerminal, NewTerm2, or over SSH to convert ye strings into ye native tongue. Run sileounpirate to revert th' effects... as long as ye backup doesn't end up in Davy Jones' Locker.

Join ye laddies!
