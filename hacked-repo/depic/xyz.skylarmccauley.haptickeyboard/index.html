xyz.skylarmccauley.haptickeyboard

Description

  • Have a haptic-enabled keyboard. Works on iOS 9 - iOS 12.1.2. Compatible with A12 as well.

  • This tweak was previously available on BigBoss. However, it was taken down (for reasons unknown to me). I decided to package the files up into DEB. I will keep this package up as long as I can. If there is an issue with this package being here, please contact me.

  • Device must already support haptics (iPhone 6S and newer, excluding the iPhone SE)

  • No preferences to configure.
  • Download DEB (Open in Safari or other internet browser)

    • Supports iOS 9 - 12.1.2
