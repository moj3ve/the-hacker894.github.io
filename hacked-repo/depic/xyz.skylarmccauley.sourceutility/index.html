xyz.skylarmccauley.sourceutility

Description

  • The all-in-one utility for managing your already added Cydia and Sileo sources.

  • Run sourceutility in the terminal to get started.

Simple Select Menu (Screenshot from v0.3)

  • Download DEB (Does not download through Cydia)

Compatibility

  • Supports iOS 9 - 12.1.2

dependencies

  • findutils

  • firmware (>= 9.0)

Conflicts

  • xyz.skylarmccauley.sourcebackup
